setup a ssh key for bitbucket

Posted on 2017-06-28

Step 1. Ensure you have an SSH client installed

$ ssh -V

OpenSSH_7.2p2 Ubuntu-4ubuntu2.2, OpenSSL 1.0.2g 1 Mar 2016

$ ssh-keygen
input passphrase

$ ps -e | grep [s]sh-agent

if the ssh-agent not running
run it
$ ssh-agent /bin/bash
add key to agent
$ ssh-add ~/.ssh/id_rsa
list out to check
$ ssh-add -l

copy public key
`cat ~/.ssh/id_rsa.pub

paset it to bitbucket or github
